# Security Policy

## Reporting a Vulnerability

Please **do not** disclose security issues publicly (issues, discussions, or social media).

Preferred reporting channels:

1. GitHub Security Advisories (if enabled for this repository), or
2. Email: security@fhir-explorer.com

When reporting, include as much detail as possible:

- Affected versions / commit SHA
- Steps to reproduce
- Impact and suggested fixes (if known)

## Supported Versions

Only the latest version on the default branch is supported.
